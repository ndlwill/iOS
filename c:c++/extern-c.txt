#if defined(__cplusplus)
#define __BEGIN_DECLS   extern "C" {
#define __END_DECLS     }
#else
#define __BEGIN_DECLS
#define __END_DECLS
#endif


__BEGIN_DECLS
extern struct __res_state *__res_state(void);
__END_DECLS


C++ 默认使用 name mangling（名字重整），会把函数名内部编码为复杂符号，例如：
int add(int, int) 可能变成 _Z3addii
而 C 是没有 name mangling 的，函数名就是原始的，如 add。
所以当头文件要同时支持被 C 和 C++ 调用时，就必须加 extern "C" 来告诉 C++ 编译器：
“这些函数是用 C 风格链接的，请不要改名字。”
