C++源代码文件的常见后缀名包括
.cpp：最常见的 C++ 源代码文件后缀。
.cc：也用于表示 C++ 源代码文件。
.cxx：另一个用于表示 C++ 文件的后缀。

C++头文件的常见后缀名包括：
.h：C++ 头文件的标准后缀名。
.hpp：也是常见的 C++ 头文件后缀，通常表示 "C++ Header"。
.hxx：另一个用于表示 C++ 头文件的后缀。
.hh：有些项目和开发者也使用 .hh 作为 C++ 头文件的后缀。
