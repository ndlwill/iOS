Global-actor isolated conformances

Implemented since Swift 6.2
Upcoming Feature Flag: InferIsolatedConformances

让 Swift 支持 协议（Protocol） 的 “隔离（isolation）” 适配（conformance），
也就是某个类型对一个协议的实现（conformance）可以被约束在某个全局 actor（global actor）之下。
就是协议适配（conformance）本身可以 “被 actor 隔离”。


Allows types isolated to a global actor (like @MainActor) to conform to protocols in a way that’s limited to their actor’s context.
For example, a @MainActor class can now conform to Equatable using an isolated conformance, which works only when called from the main actor.

现在（SE-0470）：协议 conformance 自己也可以被隔离（isolated conformance），允许类型和协议在同一个 actor 上被约束。
