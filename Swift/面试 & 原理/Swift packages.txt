Create reusable code, organize it in a lightweight way, and share it across Xcode projects and with other developers.
They bundle source files, binaries, and resources in a way that’s easy to use in your app’s project.
Xcode supports creating and publishing Swift packages, as well as adding, removing, and managing package dependencies.
Its support for Swift packages is built on top of the open source Swift Package Manager project.
