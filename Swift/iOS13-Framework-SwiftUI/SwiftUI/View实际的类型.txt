Text("Hello, World!")
    .background(Color.red)
    .frame(width: 300)
    .border(Color.gray)

在视图层级（View hierarchy）上等价于

_Overlay {
    _FrameLayout(width: 300) {
        _Background {
            Text("Hello, World!")
        }
    }
}

Overlay (gray border: RoundedRectangle)
└── Frame (width: 300)
    └── Background (Color.red)
        └── Text("Hello, World!")


虽然 SwiftUI 是闭源的，但你可以通过 SwiftUI-Introspect 或使用反射（Mirror）来窥探视图的实际结构。
https://github.com/siteline/SwiftUI-Introspect

struct ContentView: View {
    var body: some View {
        Text("Hello")
            .border(Color.red)
    }
}

在调试器（如 LLDB）中打印这个 View 的类型信息：

po Mirror(reflecting: ContentView().body)

你会看到类似 ModifiedContent<..., _OverlayModifier<...>> 的字样，说明 .border(...) 确实是通过 .overlay(...) 包装的。
