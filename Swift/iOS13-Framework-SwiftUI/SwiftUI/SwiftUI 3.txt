严格来说 SwiftUI 本身没有公开的“版本号”，不像 iOS、macOS 那样有 15、16、17 这种版本号。
我们平时说的 SwiftUI 2、SwiftUI 3 是开发者社区里的 非官方称呼，用来对应 SwiftUI 在不同 iOS 版本中引入的新功能。

大致对应关系：
SwiftUI 1.0 (2019)
随 iOS 13 / macOS 10.15 推出。
提供最基础的声明式 UI 语法、@State、@Binding、@EnvironmentObject 等。

SwiftUI 2.0 (2020)
随 iOS 14 / macOS 11 推出。
新增 LazyVGrid / LazyHGrid、ProgressView、ColorPicker、@StateObject、@AppStorage 等。

SwiftUI 3.0 (2021)
随 iOS 15 / macOS 12 推出。
新增 异步 API (.task {})、refreshable 下拉刷新、内置 markdown in Text、Searchable、AsyncImage 等。

SwiftUI 4.0 (2022)
随 iOS 16 / macOS 13 推出。
新增 NavigationStack、NavigationSplitView、Grid、Charts 等。

SwiftUI 5.0 (2023)
随 iOS 17 / macOS 14 推出。
新增 Observable 宏、@Bindable、ScrollView 改进等。


所以你看到别人说 “SwiftUI 3 引入了 markdown 支持”，意思是 在 iOS 15 引入的 SwiftUI 功能，并不是 Apple 正式给 SwiftUI 编的版本号。
要判断某个特性是在哪个“SwiftUI 版本”里加入的，其实就是查它需要的 最低 iOS / macOS 版本。
