终端 swift --version 和 swiftc --version 在命令行下通常会返回相同的版本信息

swift-driver version: 1.127.8 Apple Swift version 6.2 (swiftlang-6.2.0.13.10 clang-1700.3.13.4)
Target: arm64-apple-macosx15.0

swift 是 Swift 的交互式命令行工具（REPL），也可以用来运行 Swift 脚本。
swiftc 是 Swift 的编译器驱动程序，用于将 Swift 代码编译成可执行文件或模块。
尽管它们的用途不同，但它们底层使用的是同一个 Swift 工具链（Swift Toolchain）

命令	作用	
swift	脚本/REPL/运行环境	// 解释执行 Swift 脚本，或启动 REPL（交互式）swift repl
swiftc	编译器（Compiler）	// 编译 Swift 源码为可执行文件或目标文件 swiftc main.swift -o myProgram


// hello.swift
print("Hello, Swift!")

用 swiftc 编译并运行:
swiftc hello.swift -o hello
./hello
# 输出: Hello, Swift!

用 swift 解释运行（不生成可执行文件）:
swift hello.swift
# 输出: Hello, Swift!


REPL 是编程语言中的一个术语，全称是：
Read – Eval – Print – Loop
中文解释是：读入 – 执行 – 输出 – 循环。简单来说，它是一种交互式编程环境，让你可以一行一行地输入代码，立即看到结果。
Read（读取）：从用户输入读取一行代码
Eval（执行）：对输入的代码求值（执行它）
Print（打印）：把结果打印到屏幕
Loop（循环）：重复这个过程，直到你退出
