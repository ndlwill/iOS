iPad 使用的是 iPadOS，这是苹果专门为 iPad 设备定制的操作系统.为了让 iPad 更像电脑。

开发 iPad 应用时用的依然是 iOS SDK
Apple 实际上将 iOS 和 iPadOS 视为同一个平台（iOS Platform）。
Base SDK: iOS SDK（即使是 iPad App）

import UIKit  // 这是 iOS 的 UI 框架，也用于 iPadOS

let view = UIView()
你仍然使用 UIKit 和 iOS 提供的 API，Xcode 编译器也会标记这个 App 为运行在 iOS 平台上。
但运行时会区分是 iPhone 还是 iPad，并且你可以用如下方法做适配：
使用 UIDevice.current.userInterfaceIdiom 判断是否为 .pad
使用大小类 (Size Classes) 和 Auto Layout 实现自适应布局



你在 Xcode 中写的 App 是基于 iOS SDK 开发的，但要部署到真机或模拟器上运行，就需要对应版本的 iOS 操作系统环境支持。
iOS SDK （开发工具包）	iOS 操作系统（运行环境）
给开发者使用	给用户/设备使用
提供编译、调试、API 访问	实际运行 App 的环境
是“操作系统功能的开发接口”	是“这些功能的实际实现”

iOS 操作系统 → 一辆车（能跑、能动）
iOS SDK → 修车的说明书 + 工具（你用来造或改这辆车）


let label = UILabel()
这个 UILabel 是定义在 iOS SDK 里的一个类，运行时会映射到底层的 iOS 操作系统去显示 UI。
