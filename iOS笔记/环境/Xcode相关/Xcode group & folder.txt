当你用 Group（黄色） 添加文件时，Xcode 会在 .xcodeproj/project.pbxproj 文件里记录每一个文件节点。
所以你每新加、移动、删除一个文件，.pbxproj 都会被修改。

而当你用 Folder reference（蓝色），Xcode 不会在 .pbxproj 中列出每个子文件，
它只保存“这个文件夹的引用路径”。Xcode 在构建时会直接扫描磁盘目录下的内容。

因此：
多人协作时，使用 Folder reference 的部分不容易出现 .pbxproj 合并冲突；

Using folder references for large sets of resources can reduce merge conflicts in project files.
