Apple Clang Compiler
New Features
Clang now defines TARGET_OS_* conditionals as built-in macros based on the provided target triple. 
This extension can be disabled with the flag -fno-define-target-os-macros


target triple:
target triple 就是描述目标平台的字符串，它通常包含了下面这些信息：
架构（architecture）：如 arm64、x86_64、armv7 等
厂商（vendor）：通常是 apple、pc 等
操作系统（os）：如 ios、macosx、linux 等
环境（environment）：比如 simulator、gnu、msvc 等，有时也指 ABI 或运行时环境


iOS 真机（arm64）：
arm64-apple-ios

iOS 模拟器（x86_64）：
x86_64-apple-ios-simulator

macOS (Intel)：
x86_64-apple-macosx

macOS (Apple Silicon)：
arm64-apple-macosx


targetEnvironment(simulator)
os(iOS) 会包括 iPadOS
arch(arm64)
