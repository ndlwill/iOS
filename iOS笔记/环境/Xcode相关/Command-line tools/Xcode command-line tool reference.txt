Xcode command-line tool reference
Use command-line tools that require you to install Xcode and set the app as the active developer directory.

#####
Xcode includes a set of command-line tools that only ship with the app, such as devicectl, simctl, and xcodebuild.
You must install and set Xcode as the active developer directory before you can invoke these commands in Terminal.
#####


==================================================Automate build and version numbers
agvtool
Manage build and version numbers. To learn more about this command, enter man agvtool in Terminal.
https://developer.apple.com/library/archive/qa/qa1827/_index.html


==================================================Build a project
xcodebuild
Build Xcode projects and workspaces. To learn more about this command, enter man xcodebuild in Terminal.


==================================================Debug a project
devicectl
Manage and interact with devices connected to a host. To learn more about this command, enter xcrun devicectl help in Terminal.

xcdebug
Start a debugging session in Xcode. To learn more about this command, enter xcdebug --help in Terminal.


==================================================Edit files
xed
Open files in the Xcode app. To learn more about this command, enter man xed in Terminal.


==================================================Identify and merge changes
opendiff
Use FileMerge to graphically compare or merge files or directories. To learn more about this command, enter man opendiff in Terminal.

git difftool -y --tool=opendiff HEAD~1 HEAD -- <file>

HEAD = 当前最新提交（你现在所在 commit）
HEAD~1 = 往回一个（上一个 commit）
HEAD~2 = 往回两个 commit

为什么单独 opendiff HEAD~1 HEAD 不行？
opendiff 是“文件 diff 工具”
它不懂 Git commit hash
它只接受真实文件路径
所以必须由 Git 帮你生成两个版本的临时文件。

更简短的写法（推荐）:
先设置 opendiff 为 git 的 difftool:
git config --global diff.tool opendiff
git config --global difftool.prompt false
然后以后只需要:
git difftool HEAD~1 HEAD -- <file>
甚至：
git difftool <file>
Git 会自动用 opendiff 打开 GUI diff。

==================================================Manage scripting definition

Generate scripting definition (“sdef”). To learn more about this command, enter man desdp in Terminal.

sdef

sdp
Process scripting definition (“sdef”). To learn more about this command, enter man sdp in Terminal.


==================================================Manage the interface
actool // actool 就是 Asset Catalog 的编译和验证工具
Compile, print, update, and verify asset catalogs. To learn more about this command, enter man actool in Terminal.

ibtool // ibtool 是 Interface Builder 文件的命令行编译器和验证器
Compile, print, update, and verify Interface Builder documents. To learn more about this command, enter man ibtool in Terminal.

xcstringstool // xcstringstool 是 Xcode 的本地化字符串提取工具
Generate string tables from source code. To learn more about this command, enter xcrun xcstringstool help in Terminal.
