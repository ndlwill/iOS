英文词	中文常译	用途侧重点
credential	凭证 / 凭据 / 认证信息	表示“你是谁”的身份凭证，如用户名+密码、token、Apple ID
certificate	数字证书 / 资格证书	表示你是谁 + 你是谁说了算（由第三方签名），用于加密/验证，常用于 SSL、TLS、HTTPS


|           | credential                                                                             | certificate                                               |
| --------- | -------------------------------------------------------------------------------------- | --------------------------------------------------------- |
| 含义        | 表示“一个用户/实体的身份凭据”                                                                       | 表示“这个身份被谁信任/颁发”                                           |
| 举例        | Apple 登录时返回的 token（`ASAuthorizationAppleIDCredential`）<br>用户名+密码<br>OAuth token<br>JWT | SSL 数字证书<br>客户端证书<br>Code Signing 证书<br>Apple 的 `.cer` 文件 |
| 是否需要第三方签名 | ❌ 不需要，自己生成即可（如 token）                                                                  | ✅ 需要认证机构签名（如 CA）                                          |
| 是否用于加密    | ❌ 不直接用于加密                                                                              | ✅ 用于加密通信、公钥验证等                                            |
| 作用场景      | 登录鉴权、API 调用、身份识别                                                                       | 安全通信（HTTPS）、签名校验、App 签名、VPN                               |


credential（凭证）：你出示身份证、工牌、用户名密码，这是你自己用来证明你是谁。
certificate（证书）：你出示的身份证上有“公安机关盖的章”，这说明 你是谁 是 公安机关背书的，别人才能信。

👤 Credential 是“你是谁”的信息，
📜 Certificate 是“谁能证明你是谁”的文件。
