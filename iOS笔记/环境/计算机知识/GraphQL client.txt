Apollo iOS 是一个专门用于 GraphQL 接口通信的客户端库
它封装了请求、响应、缓存、代码生成，专注于让你轻松使用 GraphQL，而不是替代 URLSession 或 Alamofire。

GraphQL 是一种 请求协议 + 查询语言，而不是像 JSON 那样的通用数据格式。客户端会发送结构化的 GraphQL 查询字符串（Query）
比如：
query {
  user(id: "1") {
    name
    email
  }
}

服务端需要能：
解析这种查询语法
根据 Schema 查找对应的数据源
返回结构化的数据结果
进行权限校验、分页、缓存等处理

这都不是 REST API 服务器能“自然理解”的事情，必须有 GraphQL Server 做中间转换和响应。
