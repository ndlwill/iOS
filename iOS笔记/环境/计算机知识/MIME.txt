MIME（Multipurpose Internet Mail Extensions）
中文一般称为：多用途互联网邮件扩展。
MIME 是一种互联网标准，用来标识和描述电子邮件中包含的不同类型的数据格式。
它最初设计用于电子邮件系统，解决邮件只能发送纯文本的问题。
现在，MIME 被广泛用于互联网中的各种数据传输，比如 HTTP（网页传输）、邮件、文件上传下载等。

MIME 定义了内容的类型和格式，让接收方知道数据是什么类型，如何正确处理它。
| MIME 类型（Content-Type） | 说明 |
| --------------------- | ------------- |
| text/plain | 纯文本 |
| text/html | HTML 网页 |
| image/jpeg | JPEG 格式图片 |
| application/json | JSON 格式数据 |
| application/pdf | PDF 文件 |
| multipart/form-data | 表单上传（带文件等多部分） |

它帮助程序识别数据格式，实现多媒体、文本、文件等不同类型的数据传输和处理。
当浏览器请求网页时，服务器会在响应头里返回：
Content-Type: text/html; charset=utf-8
告诉浏览器，这个响应体是 HTML 格式，并且编码是 utf-8，浏览器据此正确解析渲染。
