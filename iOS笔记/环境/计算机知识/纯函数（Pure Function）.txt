纯函数具有以下两个基本特点:

1. 无副作用 (No Side Effects)：
纯函数不依赖于函数外部的状态，也不会修改函数外部的任何状态。换句话说，纯函数执行时，不会改变任何外部变量、全局变量或任何其他程序状态。
纯函数的执行仅仅依赖于传入的输入参数，并且不依赖于外部的 可变状态。

2. 输出仅依赖于输入 (Deterministic)：
纯函数的输出完全由输入决定，也就是说，对于相同的输入，纯函数每次都返回相同的输出。换句话说，无论函数执行多少次，只要输入相同，输出就会是固定的。

func add(a: Int, b: Int) -> Int {
    return a + b
}
无副作用：add 函数没有修改外部的任何状态，也没有与外部环境交互。
确定性：给定相同的 a 和 b 值，add 函数总是返回相同的结果。
