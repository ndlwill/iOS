POSIX 是 Portable Operating System Interface 的缩写，中文一般翻作 可移植操作系统接口。

POSIX 是一套 操作系统接口标准（由 IEEE 制定）。
目的是让不同的类 Unix 系统（Linux、macOS、BSD 等）在 API 层面保持兼容，这样写出来的程序更容易跨平台移植。

它规定了：
文件操作接口（open, read, write, close）
进程控制（fork, exec）
线程（pthread_* 系列）
信号（kill, signal）
错误码（errno 里的那些）

在 macOS / iOS 里，很多系统 API（尤其是网络、文件、线程相关）出错时，会返回 POSIX 定义的错误码。

POSIX = 操作系统接口标准
POSIX error = 遵循该标准的错误码
