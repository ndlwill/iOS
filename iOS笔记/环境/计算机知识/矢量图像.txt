“矢量图像”（Vector Image）是一种通过数学公式描述图形的图像类型。它与“位图图像”（Bitmap 或 Raster Image）的最大区别在于：
矢量图不是用像素点表示图像，而是用点、线、曲线、多边形等几何图元描述图像内容。


矢量图像的基本原理:
矢量图使用数学表达式定义图形。例如：
一条直线：用起点 (x₁, y₁) 和终点 (x₂, y₂) 表示；
一个圆：用圆心 (x, y) 和半径 r 表示；
一段曲线：用贝塞尔曲线控制点描述。
这些定义不依赖像素密度，因此矢量图可以无限放大而不失真。


| 特性   | 矢量图（Vector）     | 位图（Bitmap / Raster）  |
| ---- | --------------- | -------------------- |
| 构成方式 | 数学公式（线条、曲线、多边形） | 像素点（每个点都有颜色）         |
| 放大效果 | 无损、始终清晰         | 放大后会模糊、失真            |
| 文件大小 | 通常较小（取决于图形复杂度）  | 与分辨率直接相关，通常较大        |
| 适合内容 | 图标、Logo、字体、插画   | 照片、纹理、写实图像           |
| 常见格式 | SVG、PDF、EPS、AI  | PNG、JPG、BMP、GIF、TIFF |
