静态变量和全局变量在程序启动时就已经创建并初始化好了（在加载阶段就准备好了），它们的生命周期贯穿整个程序运行过程，
直到程序退出才会销毁。

编译器会把它分配在 数据段（.data） 或 BSS 段（.bss，如果未初始化）
在程序加载进内存的时候（程序还没开始执行 main()），系统已经分配好它的内存和初始值了


程序执行流程：
1. 程序加载（由操作系统负责）
可执行文件被加载进内存
操作系统根据 ELF/Mach-O/PE 文件格式中的信息，分配 .text, .data, .bss, .rodata 等段
把全局变量、静态变量放入内存，对已初始化的赋初值，未初始化的清零（.bss）

2. 运行前的初始化阶段
一些静态构造函数（如 C++ 的构造函数、__attribute__((constructor))）会在 main() 之前执行
此时你定义的 int g = 1; 或 static int x = 5; 已经都在内存中了

3. 真正执行 main()
到这个时候，你的全局变量 / 静态变量已经全部处于“可用”状态了
