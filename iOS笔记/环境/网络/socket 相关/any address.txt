/// The IPv4 any address used for listening
public static let any: IPv4Address

这里的 any 指的就是 IPv4 “通配地址”（也叫 0.0.0.0）
0.0.0.0 表示 “任意地址”，通常用在 服务器端监听。
当你创建一个 socket 并绑定到 0.0.0.0 时：
    服务器会 接受发往本机所有 IPv4 地址的连接。
    无需指定具体网卡的 IP 地址。
对客户端来说，这个地址 不能用于连接，只适合 绑定监听。


TCP/UDP 服务器监听:
let listener = try TCPListener(address: .any, port: 8080)
等价于绑定 0.0.0.0:8080。
服务器会监听所有网卡（比如 Wi-Fi、以太网）的流量。
本地多网卡适配:
假设你的电脑有多个 IP：
192.168.1.10（Wi-Fi）
10.0.0.5（以太网）
如果绑定到 .any，就可以同时监听两个地址，不需要分别绑定。
