###
macOS 默认 root 用户是禁用状态
启用 root 用户有安全风险，普通使用一般不建议启用 root。
推荐用 sudo 获取临时管理员权限。

启用并设置 root 密码
方式一：使用“目录实用工具”启用 root 用户
打开“实用工具” → “目录实用工具”
（可以通过 Spotlight 搜索 “目录实用工具”）
点击左上角的“锁”图标，输入你当前的管理员账户密码，解锁权限。
菜单栏点击“编辑” → 选择“启用 Root 用户”。
系统会提示你为 root 用户设置密码，设置后即可启用 root。
