id 是一个非常基础的 Unix/macOS/Linux 命令，它用来显示当前用户或指定用户的 用户ID（UID）、主群组ID（GID） 以及 所属群组信息（groups）。

id // 查看当前用户的信息
uid=501(alice) gid=20(staff) groups=20(staff),701(com.apple.sharepoint.group.1),...
uid=501(alice)：用户 ID 是 501，用户名是 alice。
gid=20(staff)：主群组 ID 是 20，群组名是 staff。
groups=...：列出了该用户属于的所有群组。

查看指定用户的信息：
id XXX


类型	英文术语	含义
主群组	Primary Group	用户登录时默认所属的群组，创建文件时通常作为文件的群组
所有群组	All Groups / Supplementary Groups	用户除了主群组以外，还可能属于其他群组，用于访问共享资源等
