// 生成签名密钥对
let signingKey = P256.Signing.PrivateKey()
let verifyingKey = signingKey.publicKey

// 签名
let signature = try signingKey.signature(for: Data("hello".utf8))

// 验证
let isValid = verifyingKey.isValidSignature(signature, for: Data("hello".utf8))

P256 = NIST P-256 椭圆曲线

P256.Signing
Signing 表示基于 P-256 曲线的 数字签名功能


ECDSA（数字签名）
ECDSA 全称是 Elliptic Curve Digital Signature Algorithm（椭圆曲线数字签名算法）。

它是基于椭圆曲线密码学 (ECC, Elliptic Curve Cryptography) 的一种数字签名算法，相比传统的 DSA（Digital Signature Algorithm），在相同安全强度下能使用更短的密钥长度，因此更高效、性能更好。


常见用途：
TLS/SSL 证书（比如使用 ECDSA with SHA-256）
JWT (JSON Web Token) 的签名方式 (ES256, ES384, ES512)
