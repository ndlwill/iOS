OSLog
A unified logging system for the reading of historical data.
iOS 15.0+


The OSLog framework allows you to read logs. With the unified logging system, 
you can build custom debugging and analysis tools to be used alongside Apple tools like Instruments and Console. 
To learn more about how to create logs, see Logging.
https://developer.apple.com/documentation/os/logging
